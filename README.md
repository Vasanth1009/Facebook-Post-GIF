### FBPost-GIF

App Link - [https://facebook-post-gif.netlify.app/](https://facebook-post-gif.netlify.app/)

### Steps

1. Click the input field (What's in your mind, Facebook?).
2. Type your thoughts in popup modal.
3. You can also post GIF by clicking GIF icon on bottom of modal.
4. Search GIFs and select one.
5. Click post button to post.
6. Finally, Required post will be displayed.
